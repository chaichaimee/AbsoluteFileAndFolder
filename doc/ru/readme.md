**Описание**

Перестаньте тратить время на навигацию по бесконечным каталогам. **AbsoluteFileAndFolder** — это высокопроизводительный инструмент управления, предназначенный для мгновенного и организованного доступа к вашим самым важным активам. Будь то глубоко вложенные папки проектов или критически важные системные файлы — этот аддон выводит их все на кончики ваших пальцев благодаря изысканному, но простому интерфейсу.

**Как использовать**

Добавление любимых элементов невероятно просто и интуитивно:

1. В Проводнике Windows **выделите** файл или папку, которую хотите сохранить.
2. Нажмите **горячую клавишу** (одно нажатие для папок, двойное нажатие для файлов).
3. Нажмите кнопку **Добавить** в диалоговом окне менеджера.
4. Появится окно, позволяющее задать **Отображаемое имя** или оставить оригинальное название.
5. Ваш элемент теперь надежно сохранен в списке диалога для мгновенного доступа!

**Горячие клавиши**

> **Windows + Backspace (одно нажатие):** Открывает менеджер абсолютных папок.  
> **Windows + Backspace (двойное нажатие):** Открывает менеджер абсолютных файлов.  
> **Escape (внутри диалога):** Мгновенно закрывает менеджер и возвращает вас к работе.

**Функции**

### 1. Абсолютное управление папками

Наслаждайтесь бесшовным управлением папками с инструментами, ориентированными на скорость:

- **Автоматическое сохранение:** Настройте любимые папки на автоматическое открытие при запуске Windows, продолжая работу точно с того места, где остановились.
- **Полный контроль контекста:** Щелчок правой кнопкой мыши по любой папке — закрепить наверху, изменить имя или удалить мгновенно.
- **Умная навигация:** Фокус всегда сразу попадает на список папок при открытии диалога — взаимодействие без задержек.

### 2. Абсолютное управление файлами

Точная работа с любым типом файлов в вашей библиотеке:

- **Интеллектуальная фильтрация:** Мгновенно переключайтесь между категориями Аудио, Видео, Документы, Код и **Exe**, чтобы найти нужное за секунды.
- **Инструменты для продвинутых пользователей:** Щелчок правой кнопкой по любому исполняемому файлу (Exe) — **Запустить от имени администратора** прямо из менеджера, без поиска в подменю.
- **История недавних файлов:** Никогда не теряйте из виду свою работу; аддон автоматически запоминает недавно открытые файлы в двух отдельных вкладках.

### 3. Универсальная логика управления

Оба менеджера используют мощный набор профессиональных функций:

- **Пользовательский порядок:** Перейдите в режим «Пользовательский порядок» и используйте **Вверх** / **Вниз**, чтобы расположить список именно так, как вам удобно.
- **Молниеносное удаление:** Управляйте списком уверенно — клавиша **Delete** мгновенно удаляет элемент.
- **Сортировка с учетом регистра:** Профессиональные варианты сортировки: по возрастанию с учетом регистра (a-z) и по убыванию (z-a).
- **Система закрепления:** Держите самые важные элементы всегда вверху списка, независимо от сортировки.

> **Идея в центре внимания:** Эффективность — это не только скорость, это фокус. С AbsoluteFileAndFolder ваш компьютер подстраивается под ваш рабочий процесс, а не наоборот.
