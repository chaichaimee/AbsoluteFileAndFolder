**Descripción**

Deja de perder tiempo navegando por directorios interminables. **AbsoluteFileAndFolder** es una herramienta de gestión de alto rendimiento diseñada para darte acceso instantáneo y organizado a tus activos más importantes. Ya sean carpetas de proyectos profundamente anidadas o archivos críticos del sistema, este complemento los pone todos al alcance de tu mano con una interfaz sofisticada pero sencilla.

**Cómo usar**

Agregar tus elementos favoritos es increíblemente simple e intuitivo:

1. En el Explorador de Windows, **selecciona** el archivo o carpeta que deseas guardar.
2. Pulsa la **tecla rápida** (un toque para carpetas, doble toque para archivos).
3. Haz clic en el botón **Añadir** del diálogo del administrador.
4. Aparecerá una ventana que te permitirá establecer un **Nombre para mostrar** o mantener el nombre original.
5. ¡Tu elemento ahora está almacenado de forma segura en tu lista del diálogo para acceso instantáneo!

**Teclas rápidas**

> **Windows + Retroceso (toque simple):** Abre el Administrador de Carpetas Absolutas.  
> **Windows + Retroceso (doble toque):** Abre el Administrador de Archivos Absolutos.  
> **Escape (dentro del diálogo):** Cierra inmediatamente el administrador y regresa a tu trabajo.

**Funciones**

### 1. Gestión absoluta de carpetas

Experimenta un control fluido de carpetas con herramientas diseñadas para la velocidad:

- **Persistencia automática:** Configura tus carpetas favoritas para que se abran automáticamente al iniciar Windows, continuando exactamente donde lo dejaste.
- **Control contextual total:** Clic derecho en cualquier carpeta para Anclar arriba, Editar nombres o Eliminar al instante.
- **Navegación inteligente:** El foco se coloca directamente en tu lista de carpetas cada vez que abres el diálogo, garantizando interacción sin latencia.

### 2. Gestión absoluta de archivos

Manejo preciso para cada tipo de archivo en tu biblioteca:

- **Filtrado inteligente:** Cambia instantáneamente entre categorías Audio, Video, Documento, Código y **Exe** para encontrar lo que necesitas en segundos.
- **Herramientas para usuarios avanzados:** Clic derecho en cualquier ejecutable (Exe) para **Ejecutar como administrador** directamente desde el administrador — sin más búsqueda en submenús.
- **Historial reciente:** Nunca pierdas el rastro de tu trabajo; el complemento recuerda automáticamente tus archivos accedidos más recientemente en dos pestañas dedicadas.

### 3. Lógica de gestión universal

Ambos administradores comparten un conjunto robusto de funciones profesionales:

- **Orden personalizado:** Cambia al modo "Orden personalizado" y usa **Mover arriba** o **Mover abajo** para organizar tu lista exactamente como te gusta.
- **Eliminación ultrarrápida:** Administra tu lista con confianza usando la tecla **Supr** del teclado para eliminación inmediata.
- **Ordenación por título (Title Case):** Opciones de ordenación profesionales incluyendo Ascendente sensible a mayúsculas (a-z) y Descendente (z-a).
- **Sistema de anclaje:** Mantén tus elementos esenciales siempre en la parte superior de la lista, independientemente del orden de clasificación.

> **Idea destacada:** La eficiencia no se trata solo de velocidad; se trata de enfoque. Con AbsoluteFileAndFolder, tu computadora se adapta a tu flujo de trabajo, no al revés.
