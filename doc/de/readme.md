**Beschreibung**

Hören Sie auf, Zeit mit der Navigation durch endlose Verzeichnisse zu verschwenden. **AbsoluteFileAndFolder** ist ein Hochleistungs-Verwaltungstool, das Ihnen sofortigen, organisierten Zugriff auf Ihre wichtigsten Ressourcen bietet. Egal ob tief verschachtelte Projektordner oder kritische Systemdateien – dieses Add-on bringt sie alle mit einer anspruchsvollen, aber einfachen Oberfläche direkt vor Ihre Finger.

**Verwendung**

Das Hinzufügen Ihrer Lieblingsobjekte ist unglaublich einfach und intuitiv:

1. Im Windows-Explorer **wählen** Sie die Datei oder den Ordner aus, den Sie speichern möchten.
2. Drücken Sie die **Hotkey** (Einfacher Tastendruck für Ordner, Doppeltipp für Dateien).
3. Klicken Sie im Manager-Dialog auf die Schaltfläche **Hinzufügen**.
4. Ein Fenster erscheint, in dem Sie einen **Anzeigenamen** festlegen oder den Originalnamen beibehalten können.
5. Ihr Element ist nun sicher in Ihrer Dialog-Liste gespeichert und steht für sofortigen Zugriff bereit!

**Hotkeys**

> **Windows + Rücktaste (Einfacher Tastendruck):** Öffnet den Absolute Folder Manager.  
> **Windows + Rücktaste (Doppeltipp):** Öffnet den Absolute File Manager.  
> **Escape (im Dialog):** Schließt den Manager sofort und kehrt zu Ihrer Arbeit zurück.

**Funktionen**

### 1. Absolute Ordnerverwaltung

Erleben Sie nahtlose Ordnersteuerung mit auf Geschwindigkeit ausgelegten Werkzeugen:

- **Automatische Persistenz:** Legen Sie Ihre Lieblingsordner so fest, dass sie beim Windows-Start automatisch geöffnet werden und genau dort weitermachen, wo Sie aufgehört haben.
- **Volle Kontextkontrolle:** Rechtsklick auf einen beliebigen Ordner, um ihn an den Anfang zu pinnen, Namen zu bearbeiten oder sofort zu entfernen.
- **Intelligente Navigation:** Der Fokus landet jedes Mal direkt auf Ihrer Ordnerliste, wenn Sie den Dialog öffnen – für Interaktion ohne Verzögerung.

### 2. Absolute Dateiverwaltung

Präzise Handhabung für jeden Dateityp in Ihrer Bibliothek:

- **Intelligente Filterung:** Wechseln Sie sofort zwischen Audio, Video, Dokument, Code und **Exe**-Kategorien, um in Sekunden zu finden, was Sie brauchen.
- **Power-User-Werkzeuge:** Rechtsklick auf jede ausführbare Datei (Exe), um sie direkt aus dem Manager **als Administrator auszuführen** – kein Suchen mehr in Untermenüs.
- **Verlauf der letzten Dateien:** Verlieren Sie nie den Überblick über Ihre Arbeit; das Add-on merkt sich automatisch die zuletzt verwendeten Dateien in zwei dedizierten Reitern.

### 3. Universelle Verwaltungslogik

Beide Manager teilen sich einen robusten Satz professioneller Funktionen:

- **Benutzerdefinierte Reihenfolge:** Wechseln Sie in den „Benutzerdefinierte Reihenfolge“-Modus und verwenden Sie **Nach oben** oder **Nach unten**, um die Liste genau so anzuordnen, wie Sie möchten.
- **Blitzschnelles Löschen:** Verwalten Sie Ihre Liste sicher mit der **Entf-Taste** auf Ihrer Tastatur für sofortiges Entfernen.
- **Title-Case-Sortierung:** Professionelle Sortieroptionen einschließlich Groß-/Kleinschreibung beachtend Aufsteigend (a–z) und Absteigend (z–a).
- **Pinning-System:** Behalten Sie Ihre absoluten Essentials immer ganz oben in der Liste – unabhängig von der Sortierreihenfolge.

> **Ideen-Highlight:** Effizienz geht nicht nur um Geschwindigkeit; es geht um Fokus. Mit AbsoluteFileAndFolder passt sich Ihr Computer Ihrem Workflow an – nicht umgekehrt.
