**Description**

Arrêtez de perdre du temps à naviguer dans des répertoires sans fin. **AbsoluteFileAndFolder** est un outil de gestion haute performance conçu pour vous offrir un accès instantané et organisé à vos actifs les plus importants. Que ce soient des dossiers de projets profondément imbriqués ou des fichiers système critiques, cet add-on les met tous à portée de main grâce à une interface sophistiquée mais simple.

**Comment utiliser**

Ajouter vos éléments favoris est incroyablement simple et intuitif :

1. Dans l’Explorateur Windows, **sélectionnez** le fichier ou le dossier que vous souhaitez enregistrer.
2. Appuyez sur la **touche de raccourci** (appui simple pour les dossiers, double appui pour les fichiers).
3. Cliquez sur le bouton **Ajouter** dans la boîte de dialogue du gestionnaire.
4. Une fenêtre s’ouvre vous permettant de définir un **Nom d’affichage** ou de conserver le nom original.
5. Votre élément est maintenant stocké en toute sécurité dans votre liste du dialogue pour un accès instantané !

**Raccourcis clavier**

> **Windows + Retour arrière (appui simple) :** Ouvre le Gestionnaire de dossiers Absolute.  
> **Windows + Retour arrière (double appui) :** Ouvre le Gestionnaire de fichiers Absolute.  
> **Échap (dans la boîte de dialogue) :** Ferme immédiatement le gestionnaire et revient à votre travail.

**Fonctionnalités**

### 1. Gestion absolue des dossiers

Vivez une gestion fluide des dossiers grâce à des outils conçus pour la vitesse :

- **Persistance automatique :** Configurez vos dossiers favoris pour qu’ils s’ouvrent automatiquement au démarrage de Windows, en reprenant exactement là où vous vous étiez arrêté.
- **Contrôle contextuel total :** Clic droit sur n’importe quel dossier pour Épingler en haut, Modifier le nom ou Supprimer instantanément.
- **Navigation intelligente :** Le focus se pose directement sur votre liste de dossiers à chaque ouverture du dialogue, pour une interaction sans latence.

### 2. Gestion absolue des fichiers

Gestion précise pour chaque type de fichier dans votre bibliothèque :

- **Filtrage intelligent :** Basculez instantanément entre les catégories Audio, Vidéo, Document, Code et **Exe** pour trouver ce dont vous avez besoin en quelques secondes.
- **Outils pour utilisateurs avancés :** Clic droit sur tout exécutable (Exe) pour **Exécuter en tant qu’administrateur** directement depuis le gestionnaire — plus besoin de fouiller dans les sous-menus.
- **Historique récent :** Ne perdez plus la trace de votre travail ; l’add-on se souvient automatiquement de vos fichiers les plus récemment accédés dans deux onglets dédiés.

### 3. Logique de gestion universelle

Les deux gestionnaires partagent un ensemble robuste de fonctionnalités professionnelles :

- **Ordre personnalisé :** Passez en mode « Ordre personnalisé » et utilisez **Déplacer vers le haut** ou **Déplacer vers le bas** pour arranger votre liste exactement comme vous le souhaitez.
- **Suppression ultra-rapide :** Gérez votre liste en toute confiance avec la touche **Suppr** de votre clavier pour une suppression immédiate.
- **Tri en casse de titre :** Options de tri professionnelles incluant Ascendant sensible à la casse (a-z) et Descendant (z-a).
- **Système d’épinglage :** Gardez vos essentiels absolus toujours en haut de la liste, indépendamment de l’ordre de tri.

> **Mise en lumière de l’idée :** L’efficacité ne se limite pas à la vitesse ; elle concerne la concentration. Avec AbsoluteFileAndFolder, votre ordinateur s’adapte à votre flux de travail, et non l’inverse.
