## Опис
Досить витрачати час на навігацію нескінченними папками. **Апсолютні папки та файли** — це потужний інструмент управління для миттєвого доступу до ваших ресурсів.

## Як використовувати
1. У Провіднику Windows **виберіть** файл або папку.
2. Натисніть **гарячу клавішу**.
3. Натисніть кнопку **Додати**.
