## Beschreibung
Verschwenden Sie keine Zeit mehr mit der Navigation durch endlose Verzeichnisse. **AbsoluteFileAndFolder** ist ein Hochleistungs-Verwaltungstool, das Ihnen sofortigen und organisierten Zugriff auf Ihre wichtigsten Ressourcen bietet.

## Bedienung
Das Hinzufügen Ihrer Favoriten ist unglaublich einfach und intuitiv:

> [!TIP]
> 1. Wählen Sie im Windows-Explorer die Datei oder den Ordner aus, den Sie **speichern** möchten.
> 2. Drücken Sie die **Tastenkombination** (einmal drücken für Ordner, zweimal drücken für Dateien).
> 3. Klicken Sie im Verwaltungsdialog auf die Schaltfläche **Hinzufügen**.
> 4. Ihr Element ist nun sicher in Ihrer Liste gespeichert!

## Tastenkombinationen
- **Windows + Rücktaste (einmal drücken):** Öffnet den Absolute Folder Manager.
- **Windows + Rücktaste (zweimal drücken):** Öffnet den Absolute File Manager.

## Funktionen
### 1. Absolute Ordnerverwaltung
* **Automatischer Start:** Konfigurieren Sie Ihre Favoriten so, dass sie beim Windows-Start automatisch geöffnet werden.
* **Volle Kontextkontrolle:** Rechtsklick auf einen Ordner zum Oben anheften, Umbenennen oder sofortigen Löschen.

### 2. Absolute Dateiverwaltung
* **Intelligente Filterung:** Wechseln Sie sofort zwischen den Kategorien Audio, Video, Dokument, Code und **Exe**.
* **Power-User-Tools:** Rechtsklick auf eine ausführbare Datei (Exe), um sie direkt als **Administrator auszuführen**.

> [!IMPORTANT]
> **Kernidee:** Effizienz bedeutet nicht nur Geschwindigkeit, sondern Fokus.
