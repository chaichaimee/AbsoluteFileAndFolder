## Описание
Хватит тратить время на навигацию по бесконечным папкам. **AbsoluteFileAndFolder** — это мощный инструмент управления для мгновенного доступа к вашим самым важным файлам.

## Как использовать
1. В проводнике Windows **выберите** файл или папку.
2. Нажмите **горячую клавишу** (одно или два нажатия).
3. Нажмите кнопку **Добавить**.
