## Description
Arrêtez de perdre du temps à naviguer dans des répertoires sans fin. **AbsoluteFileAndFolder** est un outil de gestion haute performance conçu pour vous offrir un accès instantané et organisé à vos dossiers les plus importants.

## Comment utiliser
1. Dans l’Explorateur Windows, **sélectionnez** l’élément.
2. Appuyez sur la **touche de raccourci** (simple ou double appui).
3. Cliquez sur **Ajouter**.
