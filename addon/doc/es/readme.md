## Descripción
Deja de perder el tiempo navegando por directorios interminables. **AbsoluteFileAndFolder** es una herramienta de gestión de alto rendimiento diseñada para ofrecerte un acceso instantáneo y organizado a tus activos más importantes.

## Cómo utilizar
Añadir tus elementos favoritos es increíblemente sencillo e intuitivo:

> [!TIP]
> 1. En el Explorador de Windows, **selecciona** el archivo o carpeta.
> 2. Presiona la **tecla de atajo** (un toque para carpetas, dos toques para archivos).
> 3. ¡Tu elemento ya está guardado de forma segura!

## Atajos de teclado
- **Windows + Retroceso (un toque):** Abre el Gestor de Carpetas Absolute.
- **Windows + Retroceso (dos toques):** Abre el Gestor de Archivos Absolute.
